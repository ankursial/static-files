zz_Overview_ZZ

zz_This step extracts the content of a raw document into a set of filter events._ZZ

zz_Takes: Raw document
Sends: Filter events_ZZ

zz_This step is used to take a raw document with an associated filter configuration and send the filter events down the pipeline. It basically extract the content and structure information of the input so steps that take filter events as input can be used. This step is generally the first one of the pipeline, but it can be placed after any step that sends a raw document._ZZ

zz_To merge back filter events into a raw document use the Filter Events to Raw Document step._ZZ

zz_Parameters_ZZ

zz_This step has no user parameters. All the parameters are provided by the calling application._ZZ
